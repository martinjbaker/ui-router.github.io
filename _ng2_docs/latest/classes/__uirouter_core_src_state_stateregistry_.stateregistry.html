onStatesChanged

  • onStatesChanged(listener: StateRegistryListener): () => void
    • Defined in @uirouter/core/src/state/stateRegistry.ts:101

    Listen for a State Registry events

    Adds a callback that is invoked when states are registered or deregistered with the StateRegistry.

    Example:

     
    let allStates = registry.get();

// Later, invoke deregisterFn() to remove the listener
let deregisterFn = registry.onStatesChanged((event, states) => {
  switch(event) {
    case: 'registered':
      states.forEach(state => allStates.push(state));
      break;
    case: 'deregistered':
      states.forEach(state => {
        let idx = allStates.indexOf(state);
        if (idx !== -1) allStates.splice(idx, 1);
      });
      break;
  }
});

    Parameters

    • listener: StateRegistryListener

      a callback function invoked when the registered states changes. The function receives two parameters, event and state. See StateRegistryListener

    Returns () => void

    a function that deregisters the listener

      • Returns void
